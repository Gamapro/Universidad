Examen 2do Parcial

Realizar las siguientes paginas web con uso de HTML, CSS y JavaScript

CALCULADORA

Dados los archivos HTML y CSS realizar el funcionamiento de la calculadora mediante JS. la calculadora debera realizar todas las operaciones que una calculadora basica en vida real

TODO LIST

Se debera realizar una pagina que funcione como0 una lista de recordatorios(todo-list) en la cual se pueda ingresar mediante un campo de texto la accion o producto a recordar y este se anexara a una lista la cual sera visible y modificable

QUIZ

Realizar un juego de tipo cuestionario(quiz) el cual constara de 10 preguntas en el especto de programacion. se llevara un registro de las preguntas respondidas y al final se entregara un puntaje total de el usuario que haya respondido el quiz
